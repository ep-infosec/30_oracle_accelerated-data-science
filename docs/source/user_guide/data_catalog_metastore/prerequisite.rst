.. include:: ../_template/prerequisite/data_flow.rst

To configure Data Catalog you will need to:

.. include:: ../_template/prerequisite/data_catalog.rst
* The ``spark-defaults.conf`` file needs to be :ref:`configured<configuration-spark_defaults_conf>`.
